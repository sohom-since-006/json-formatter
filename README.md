# JSON Formatter
A simple JSON formatter that takes an unstructured and unformatted JSON text and returns the formatted JSON.

---

## Project Overview

**Tech Stack:**
- Frontend: React (Vite)
- Backend: Node.js + Express
- Storage: In-memory (No database)

**Current features:**
- Input an unstructured JSON
- Returns a formatted JSON

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/devsoc-aec/json-formatter.git
```

### 2. Install dependencies
**Server**
```bash
cd server
npm install
npm run dev
```

**Client**
```bash
cd client
npm install
npm run dev
```
